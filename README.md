# RAG для Яндекс ML Handbook

Retrieval-Augmented Generation система для поиска и ответов на вопросы по материалам [Яндекс ML Handbook](https://education.yandex.ru/handbook/ml).

## О проекте

Цель — создать чат-бота, который отвечает на вопросы по машинному обучению, используя контент из учебника Яндекса как базу знаний.

## Что сделано

- [x] **Сбор данных** — парсинг 69 статей из ML Handbook с помощью Selenium
- [x] **Предобработка** — очистка HTML, удаление мусора, нормализация текста
- [x] **Чанкирование** — разбиение документов на фрагменты для векторного поиска

## Структура проекта

```
├── data/                    # HTML-файлы статей
├── data_processing/
│   ├── html_parser.py       # Скрапер для загрузки статей
│   └── data_utils.py        # Утилиты обработки данных
├── show_preprocessing.ipynb # Демонстрация препроцессинга
└── data.zip                 # Архив с данными
```

## Установка

```bash
pip install langchain langchain-community beautifulsoup4 selenium
```
